## 项目
基于vue-cli3.0，vue，ts开发的一款pwa应用。

![show](https://raw.githubusercontent.com/HolyZheng/holyZheng-blog/master/images/beshow.gif)

## 为什么做这个项目
- pwa 在国内已经火过一段时间了，但是自己还没做过一款pwa应用。
- vue-cli 3.0 增加了对pwa的支持
- vue2.5后增加了对ts的支持
- 想搞事情！
## 项目要点
- 在vue中使用ts
- 简单骨架屏的运用
- 首屏加载时间和seo的优化
- pwa相关特性的实现
- 移动端的一些问题解决
- 如何部署项目
